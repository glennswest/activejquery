activejquery by glennswest

A set of Jquery Tools for Restful Jquery Usage

ActiveJquery combines the following into one plugin 1. JqGrid Based Data Tables 2. Jquery UI for theming 3. Jquery UI For Menu Tabs The plugin provides restful controller logic, dynamically generated javascript for jqgrid, and a easily entensible menu system. The goal is to provide a complex application client that is Ajax streamed to the browser, with little coding on the part of the user.
